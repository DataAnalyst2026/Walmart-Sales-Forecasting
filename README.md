# Walmart Sales Forecasting

## Overview
This project predicts Walmart weekly sales using time series forecasting techniques.

## Objective
Forecast future weekly sales for each Walmart store to support inventory and business planning.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels (SARIMA)
- Jupyter Notebook

## Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Time Series Forecasting
- Model Evaluation
- Sales Forecast Visualization

## Results
Developed a forecasting model to predict future weekly sales for Walmart stores using historical sales data.
